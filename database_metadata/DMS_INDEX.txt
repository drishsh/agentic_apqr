SEARCH TIPS FOR AGENTS:
================================================================================

**FINDING SOPs:**
- Current Version: Look in "13. List of all the SOPs/Version-2/" (most recent)
- Previous Version: Look in "13. List of all the SOPs/Version-1/"
- SOP Naming: "SOP-[DEPT]-[NUMBER]_[Title].pdf"
- Departments: MFG (Manufacturing), QC (Quality Control), QA (Quality Assurance), 
              ENG (Engineering), WH (Warehouse), HR (Human Resources)

**FINDING TRAINING RECORDS:**
- Training Matrices: "14. Comprehensive_training_records/training_matrices/"
- File Pattern: "[Department]_Training_Matrix_2025.xlsx"
- Departments: Manufacturing, QC, QA, Engineering, Warehouse

**FINDING CAPA DOCUMENTS:**
- Folder: "CAPA Documents/"
- File Pattern: "CAPA_[NUMBER]_[Issue]_[Details].pdf"
- Batches Referenced: ASP-25-001, ASP-25-002, ASP-25-003, ASP-25-004
- CAPA Register: "CAPA_Register_2025.xlsx" (complete list)

**FINDING REGULATORY DOCUMENTS:**
- SDS Register: "Master_SDS_Register.xlsx"
- Product Specifications: Search for "Product_Specification_[Product].pdf"
- Regulatory Files: Search for "Drug_Master_File" or "Regulatory_Submission"

**DOCUMENT TYPES:**
- SOPs: Standard Operating Procedures
- CAPA: Corrective and Preventive Action
- SDS: Safety Data Sheet
- DMF: Drug Master File
- GMP: Good Manufacturing Practice

**KEY SEARCH TERMS:**
- For procedure questions: Search SOPs by department code
- For training questions: Check training matrices
- For quality issues: Check CAPA documents
- For safety: Check SDS documents
- For regulatory: Check DMF and product specifications
