SEARCH TIPS FOR AGENTS:
================================================================================

**FINDING COAs FOR RAW MATERIALS:**
- Batch 1: "COA_[Material].pdf" in folder 01. Aspirin_Procurement_Details/
- Batch 2-4: "COA_[Material]_ASP-25-00X.docx" in folder 01. Aspirin_Procurement_Details/
- Materials: API, Binder, Diluent, Disintegrant, Lubricant

**FINDING FINISHED PRODUCT COAs:**
- Look in "04. Internal QC Register & COA/" folder
- File: "Certificate of Analysis_Batch_ASP-25-00X.pdf"

**FINDING IPC (IN-PROCESS CONTROL) DATA:**
- Compression IPC: Look in "07. Compression/" or "04. Internal QC Register & COA/"
- Coating IPC: Look in "04. Internal QC Register & COA/"
- File pattern: "IPC Checks During [Process]-ASP-25-00X.pdf"

**FINDING SAMPLING RECORDS:**
- Folder: "03. Sampling Booth Logs/"
- Files: "Sampling Booth (SB-01)" or "Sampling Booth (SB-02)"

**FINDING QUARANTINE LOGS:**
- Folder: "02. APQR_Quarantine and GRN Logs/" or "09. Sampling And Quarantine/"
- Files: "Quarantine Register.pdf", "Quarantine Daily Verification Log"

**FINDING PACKAGING MATERIAL QC:**
- Folder: "08. Packaging_Materials_Procurement/"
- Files: "COA_Packaging_[Material].pdf"

**MATERIAL MAPPING:**
- API = Salicylic Acid (Active Pharmaceutical Ingredient)
- Binder = HPMC (Hydroxypropyl methyl cellulose)
- Diluent = MCC (Microcrystalline Cellulose)
- Disintegrant = Cornstarch
- Lubricant = Magnesium Stearate

**BATCH IDENTIFICATION:**
- Batch 1 = ASP-25-001 (Jan-Feb)
- Batch 2 = ASP-25-002 (Feb-Mar)
- Batch 3 = ASP-25-003 (Mar-Apr)
- Batch 4 = ASP-25-004 (Apr-May)

**TEST PARAMETERS IN COAs:**
- Appearance: Visual description
- Identification: FTIR, Chemical tests
- Assay: Active content percentage
- Impurities/Related Substances: Purity testing
- Physical Tests: Weight, Thickness, Hardness, Friability, Disintegration, Dissolution
- Moisture Content/LOD: Loss on Drying
- Heavy Metals: Contamination testing
- Microbial Limits: Bioburden testing
